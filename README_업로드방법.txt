GitHub Pages 업로드 방법

1. 저장소의 기존 파일을 전부 삭제하세요.
2. 이 ZIP을 압축 해제하세요.
3. 압축 해제된 내용물(index.html, styles.css, favicon.png, assets, ruffle, .nojekyll)을 저장소 루트에 그대로 업로드하세요.
4. ZIP 파일 자체를 업로드하면 실행되지 않습니다.
5. 배포 후 Ctrl+F5로 강력 새로고침하세요.

정상 업로드 확인:
- GitHub 저장소에 assets/game.swf 파일이 보여야 합니다.
- GitHub 저장소에 ruffle/ruffle.js 파일이 보여야 합니다.
